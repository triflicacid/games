# Games

This repository holds (small) games that I have created.

To host, run `npm install` and execute `npm start` (runs `http-server`). Some games do not require a server, and may be run by double-clicking on `index.html`

## Included Games
- Chopsticks
- Clicker
- Connect 4
- Marble Solitaire
- Minesweeper
- Reaction Time
- Snake
